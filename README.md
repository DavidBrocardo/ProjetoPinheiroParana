# Projeto Pinheiro do Paraná 🌲

O **Projeto Pinheiro do Paraná** é uma iniciativa dedicada à conservação e preservação da *Araucaria angustifolia* (Pinheiro-do-Paraná), espécie símbolo da região Sul do Brasil e que se encontra em risco crítico de extinção. 

Este projeto atua como um portal e sistema web cujo principal objetivo é promover a **educação ambiental**, divulgar a **pesquisa científica** e fomentar **ações de extensão** relacionadas à espécie e ao seu ecossistema associado (Floresta Ombrófila Mista).

## 🎯 Objetivo do Projeto

Na área de divulgação científica, o portal visa:
- **Democratizar o Conhecimento:** Traduzir dados científicos e pesquisas sobre a Araucária para uma linguagem acessível ao público geral.
- **Educação Ambiental:** Sensibilizar a comunidade sobre a importância ecológica da árvore e da fauna associada (como a Gralha-azul).
- **Apoio à Pesquisa e Extensão:** Servir como plataforma centralizadora para notícias, galerias de fotos históricas e atuais, e o catálogo da equipe de pesquisadores e voluntários envolvidos na preservação.
- **Engajamento:** Incentivar a participação da sociedade em ações de conservação através de um design imersivo e acolhedor.

## 🛠️ Tecnologias Utilizadas

Para garantir que a plataforma seja não apenas funcional, mas também proporcione uma experiência de usuário (UX) premium e de alta performance, utilizamos tecnologias modernas de desenvolvimento web:

*   **React 19 (com TypeScript):** Biblioteca base para a construção da interface de usuário, garantindo código tipado, seguro e componentes reutilizáveis e modulares.
*   **Vite:** Ferramenta de build super rápida, que otimiza o ambiente de desenvolvimento e a compilação do projeto para produção.
*   **React Router DOM v7:** Gerenciamento eficiente de rotas, permitindo navegação fluida ("Single Page Application") entre as diversas seções do portal sem recarregar a página.
*   **Firebase (Firestore & Storage):** Infraestrutura em nuvem utilizada como banco de dados em tempo real (para gerenciar notícias, galeria e equipe) e armazenamento seguro de arquivos e imagens, facilitando a atualização dinâmica de conteúdo sem necessidade de deploys constantes.
*   **Framer Motion:** Biblioteca responsável por trazer o portal "à vida" com animações suaves e transições dinâmicas baseadas no scroll, aumentando o engajamento do usuário.
*   **Swiper.js:** Implementação de carrosséis e sliders responsivos para exibição imersiva de imagens e conteúdos em destaque.
*   **Vanilla CSS:** Estilização desenvolvida com CSS puro, focada em performance e manutenção, organizada de forma modular e aplicando conceitos de UI/UX modernos (como Glassmorphism e paleta de cores naturalista).
*   **React Icons:** Biblioteca de ícones vetoriais leves e escaláveis para melhorar a sinalização visual das interfaces.


## 🚀 Como Executar o Projeto Localmente

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente do Firebase no arquivo `.env.local` (baseando-se no `.env.example`).
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
